NASA's Astronomy Picture of the Day Website Revamped

I don't know if you also feel the pain when browsing the Astronomy Picture of the Day website of NASA. Although I love the beautiful pictures curated and its in-depth explanation by professional astronomers, the interface and layout feel dated. As a personal project I tried to redesign and revamp part of the site with modern web standards. The idea is to reorganize the user interface and improve site navigation. I focused on showcasing the daily picture by making good use of the browser real estate and improving navigation flow. The site was built with AngularJS, jQuery, NASA APOD API, Youtube API, and other JavaScript plugins.

My Community

This is a simple web application showing the heatmap of crime incidents over the past year around an user specified address. If crime data exist, a slidebar will appear, allowing users to scroll and see the trend of crime frequency changing over time. Map data is from Google Maps API; crime data is from Crime Data API on Mashape. Application is built with Knockout.js and jQuery. [Note] The Crime Data API is currently having some trouble; we will be looking for other data source and update our app asap.

Help Me Choose

As someone who just can not decide what to eat for lunch every day, a web application that generates random results is life (or time) saver. This application has a variety of pre-defined lists. Users can also draw from his or her own custom list. It is also capable of producing random numbers from a user defined range, very helpful when buying lottery. A Completely New Dynamic Website for ASIAA

Academia Sinica Institute of Astronomy & Astrophysics (ASIAA) is an academic institute where I worked as a webmaster several years ago. To be able to present the many research projects, groups, astronomy gallery, press release, information for colloquium and seminars of the institute, the site needed a database to hold the vast amount of information, and dynamic templates to be manageble and scalable. They also needed an internal platform for managing member's travel schedule and publication records. As a result, I rebuilt the originally static web site to a dynamic one with PHP and MySQL. The current interface was redesigned by successive webmasters. However, most of the base structrue, the UI logic, and the codes behind the scene remain the same.
